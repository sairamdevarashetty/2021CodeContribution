#FILEMERGE


USAGE: Helps to merge the multiple pdf files in each subdirectory to single 

COMMAND: python parentrepo

Example:
   parentrepo(dir)
       childrepo1(dir)
          one.pdf
          two.pdf
       childrepo2(dir)
          three.pdf
          four.pdf


output
  chilrepo1.pdf (includes both one and two)
  childrepo2.pdf (includes both three and four)
